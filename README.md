### Описание проекта

Консольное приложение для управления списком контактов. Реализовано в рамках первого модуля обучения с акцентом на чистый функциональный подход без использования ООП.

**Основные возможности:**

* **CRUD**: Создание, чтение, обновление и удаление контактов.
* **Data Validation**: Проверка корректности Email, длины телефона (12 цифр) и уникальности данных.
* **Smart Search**: Поиск по имени или номеру телефона (через префикс `+`).
* **Storage**: Работа с текстовой базой данных (`contacts.txt`) с сохранением состояния.
* **UX**: Глобальные команды `exit` и `menu` доступны в любом контексте ввода.

### Технический стек

* **Язык:** Python 3.10+
* **Концепции:** Функции высшего порядка, лямбда-сортировка, валидаторы, работа с файловой системой.
* **Инструменты:** Git.

### Мета-информация

* **Время работы:** 6 часов.
* **Контекст:** Код написан с соблюдением учебных ограничений (без классов), что потребовало сложной архитектуры взаимосвязанных функций.
